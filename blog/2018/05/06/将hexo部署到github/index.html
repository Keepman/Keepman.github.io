将hexo部署到github

发表于 2018-05-06
  1. 首先我们在github上新建一个项目仓库，必须取名为username.github.io

  2. 在hexo项目的根目录下的_config.yml中需要更改配置

    //需要注意的是这些冒号后面都需要跟上空格
deploy: 
  #我们是使用 git 来部署的。
  type: git
  # git 仓库的地址。
  repo: <仓库地址>
  # 分支名称。
  branch: master
  3. 上传项目至github
  4. 执行命令hexo clean && hexo g && hexo d
